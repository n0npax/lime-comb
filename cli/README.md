# Lime-comb

This tool allows:
-   Encrypt messages directly from your terminal.
-   Push & pull keys to central registry

## encrypt

Example usage:

```bash
$lime-comb enc -t ${SOMEONES_EMAIL} -m "db_qa_password: SUPERSECRET1" -m "dq_preprod_password:SUPERSECRET2"
```
